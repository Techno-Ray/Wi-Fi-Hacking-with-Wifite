Automated Wi-Fi Hacking with Wifite

Hello Rays of Tech,
In this tutorial we are going to look for wifi hacking using the inbuild tool of Kali Linux called Wifite.

For this tutorial we will use Command line interface (CLI). There is also an Graphical user interface (GUI) if you want you can also use GUI which is a very user friendly interface.

But, for this tutorial I am going to assume that you are just a noob into this field and doesn’t know anything about CLI (Command line interface). Evevn the CLI of Wifite is very similar to as GUI so dont worry!!!

Lets Start Hacking “ETHICLLY”:

We will move stepwise further so you will not get loost in the “Terminal” i.e. Command line.

Before starting the tools for the Pentest let’s have a look around the things we will need in this tutorial. You will need the things listed below:

1) Open Source Linux opreating System. (“Kali linux/Parrot Os” Recommended).

2) Wi-Fi Adapter supporting monitor mode and Packet Injection. (In my case I am using TP-link 150Mbps High gain V3.)



3) Get Close to the Target as possible as you can. (Advice:-Please be a little smart.)
4) A little bit of free Time.
5) Let’s Get Started…

Follow Completely and correctly and if you are unsure just Copy and Paste Commands.


Steps to Get Started:

1) Start your Linux OS.

In my Case it’s Kali-Linux.

2) Open up your terminal from upper left corner if not present just type “ctrl+alt+T” on your Desktop. This will open your Terminal.

3) Now Type or just Copy and Paste All the Commands directly.

   sudo apt update     (to update System)

   sudo apt upgrade    (to upgrade System)

4) Now we have to put our Wireless card into Monitor Mode. Let’s first check if the card is recognised by the linux box or Machine. Type the command

 iwconfig

and check if you have got the wlan interface. If present Move Forward.

5) Now let us put the Wlan (wifi card) into the monitor mode by following commands:

                 sudo airmon-ng check kill

                  sudo airmon-ng start [INTERFACE NAME]

This will put your Wireless interface into Monitor mode. You can cross-check it using command “iwconfig”.
Note, this will switch off your internet connection on that Interface.

6) Now let’s start the tool Wifite by using Command

   sudo wifite

7) This will automatically start scanning the Wi-Fi Network around you.
8) Now when you identifies the Target name just Press “ctrl+C” on your Keyboard. This will stop scanning the Networks and will show you the options to select Target network to start attack.

In this Tutorial we are going to use “Dictionary Attack” So…

9) Press “ctrl+c”
10) Then It will ask you if you want to continue to the next attack Press “c” to continue attacking

11) Do it Twice Again and Follow to the Fourth attack when it says that (Capturing Handshake) and stop and let it Go Forward on it’s own.
12) this will now Deauthentic all the Users Connected to the Target Network, and when the devices will try to connect again Wifite will automatically capture the HandShake and will save it to the /hs directory. And also will automatically start the dictionary attack.
